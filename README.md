# Bart-Summarization
Summarization Task using BART models from [HugginFace Transformers](https://github.com/huggingface/transformers)


## Install requeriments
```
pytorch
docx2txt==0.8
Flask==2.0.1
transformers==4.8.2
Werkzeug==2.0.1
```

## Run
```
python application.py
```
